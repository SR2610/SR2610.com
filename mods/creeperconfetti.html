Creeper Confetti

Creeper Confetti is a mod for the game Minecraft that replaces the default creeper explosion with harmless confetti.

Project Description

Creeper Confetti was originally created in July 2014 and has been maintained since with updates for every Minecraft version since release with other maininence updates inbetween.

The mod has since found its way into over 600 modpacks with over 5 Million Downloads between versions.

Project Details

  • Game: Minecraft Java Edition
  • Initial Release Date: July 2014
  • Total Downloads: 5 Million+
  • Code: Github
  • Download: CurseForge
