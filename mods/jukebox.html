Jukebox

Jukebox is a mod for the game Minecraft that adds a more advanced version of the Jukebox block in game with the ability to store records and play them.

PreviousNext

Project Description

Jukebox was originally created in June 2017 and has been sporadically updated and maintained since release.

The mod has since been featured in over 40 modpacks with 30,000 downloads between versions.

Project Details

  • Game: Minecraft Java Edition
  • Initial Release Date: June 2017
  • Total Downloads: 30,000+
  • Code: Github
  • Download: CurseForge
