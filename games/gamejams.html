Game Jams

All of the games on this page were made for Game Jams of varying lengths.


The Bone Zone

In this game you roll around as a skull, picking up other bones to put your skeleton back together.

This project was created over the course of three hours as part of Solent Spooky Jam. The idea for the game was based off of Katamari Damacy.

  • Role: Developer

Waist of Time

Waist of Time is a 2D Sidescrollling Platformer where the player needs to use time manipulation mechanics in order to solve puzzles.

PreviousNext

Project Description

This game was built over 5 days as part of a Southampton Solent Game Jam with the theme "Time". The game uses a collection of time manipulation abilities such as rewinding, fast-forwarding, and skipping time in order to solve puzzles in the platforming.

Project Details

  • Game Engine: Unity
  • Platform: PC
  • Role: Developer
  • Collaborators: Jaz Marshall-Porter, Timothy Bush, Mitchell Dixon, Rob Ashton

Project Learnings

This project was one of the first times I worked in team with the Unity Engine and was useful for understanding issues with the Unity file system and version control systems such as Git.
